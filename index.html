HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer element contains information such as author.
  • The span element provides an inline space.
  • The class and id elements are used to help group elements together. This can help with styling and adding functions in JavaScript.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Classes are used to assign styles to particular elements.
  • The . selector is used to select classes and the # selector is used to select by id. The * is used to select all elements.
  • The text-align attribute determines the alignment of text
  • The border element puts a line around the element, you specify its thickness in pixels(px).

Git

  • The git cheat sheet is a good reference for git commands that can be used in the command line (GitBash). The Git workflow video is a useful reference for understand how to upload and download changes to and from Git.
  • I should work on repositories cloned from GitHub locally on a new feature branch. I should always pull the most recent main branch from Git before committing any changes from my feature branch.
  • git status: tells you which github branch you're in.
  • git checkout -b branch-name: creates a new branch.
  • git add -A: adds a new branch to a working branch.
  • git commmit 'm ("tailor your message here"): adds a message that describes what the commit includes.
  • git pull origin main: checks local branch is in sync with the base branch in GitHub.
  • git push origin branch-name: pushes changes to the remote branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions enable us to store blocks of code that can be reused. Functions must be called before they execute code.
  • Conditionals (if statements) run as soon as the JavaScript file is open unless stored in a function.
  • An array groups multiple values together. They can be the same or different data types.
  • A for loop allows us to loop over arrays to retrieve individual values.
